Module: Open Web Analytics


Description
===========
Adds the Open Web Analytics tracking system to your website.

Requirements
============

* Open Web Analytics installation
* Open Web Analytics SiteID


Installation
============
* Copy the 'owa' module directory in to your Drupal
sites/all/modules directory as usual.


Usage
=====
In the settings page enter your Open Web Analytics SiteID and the Open Web
Analytics installation url.

All pages will now have the required JavaScript added to the
HTML. You can confirm this by viewing the page source from
your browser.
